#### Insertar código
En consola (símbolo del sistema):

  ```console
  $ pip install python
  ```

  ... código en python:
  ```python
  print ("Hello world")
  ```

  ...o código en java (Arduino):
  ```java
  void setup() {
    pinMode(13,OUTPUT);
    }
  void loop() {
    digitalWrite(13,HIGH);
  }
  ```

* ...o HTML

  ```html
    <html>
      <body>
        Hello world!!
      </body>
    </html>
  ```
  * ...o Processing:
  ```processing
  void setup() {
    size(400,200);
    }
  void draw() {
    background(0);
  }
  ```

#### Insertar código
En consola (símbolo del sistema):
* Código en python:
  ```python
  print ("Hello world")
  ```
* Código en java (Arduino):
  ```java
  void setup() {
    pinMode(13,OUTPUT);
    }
  void loop() {
    digitalWrite(13,HIGH);
  }
  ```
* Código en HTML
  ```html
    <html>
      <body>
        Hello world!!
      </body>
    </html>
  ```
* Código en Processing:
  ```processing
  void setup() {
    size(400,200);
    }
  void draw() {
    background(0);
  }
  ```
